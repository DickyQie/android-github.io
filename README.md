# AndroidStudio安装教程(Windows环境下)
 <span id="OSC_h3_1"></span>
<h3>AndroidStudio官网下载：<a href="http://android-studio.org/" rel="nofollow">http://android-studio.org/</a>&nbsp; &nbsp; 可以更具自己喜欢的版本下载，个人推荐2.2版本以上，因为开发和运行效率快，高很多。</h3> 
<span id="OSC_h3_2"></span>
<h3>Android Studio的Gradle配置文件下载：<a href="http://services.gradle.org/distributions/" rel="nofollow">http://services.gradle.org/distributions/</a>&nbsp; 包含多个版本</h3> 
<span id="OSC_h3_3"></span>
<h3>要安装Android Studio并使用的话先要配置Java环境变量， 请看 &nbsp; <a href="https://my.oschina.net/zhangqie/blog/778364" rel="nofollow">Java环境变量配置&nbsp;</a></h3> 
<span id="OSC_h4_4"></span>
<h4>下载会有两种：&nbsp;zip包和.exe程序，zip包则解压即可 &nbsp;效果图：</h4> 
<p><img alt="" height="87" src="https://static.oschina.net/uploads/space/2016/1226/141943_lOmu_2945455.png" width="543"></p> 
<span id="OSC_h4_5"></span>
<h4>打开文件夹，找到bin目录下的 studio64 &nbsp;双击打开，直接下一步就可以了</h4> 
<p><img alt="" height="188" src="https://static.oschina.net/uploads/space/2016/1226/142605_Hhgk_2945455.png" width="527"></p> 
<span id="OSC_h4_6"></span>
<h4>这是用于导入Android studio的配置文件，如果是第一次安装，选择最后一项：不导入配置文件，然后点击OK即可。</h4> 
<p><img alt="" height="123" src="https://static.oschina.net/uploads/space/2016/1226/143539_ZeNE_2945455.png" width="287"></p> 
<span id="OSC_h4_7"></span>
<h4>点击Setup Proxy&nbsp;</h4> 
<p><img alt="" height="380" src="https://static.oschina.net/uploads/space/2016/1226/144132_meDf_2945455.png" width="534"></p> 
<p><img alt="" height="397" src="https://static.oschina.net/uploads/space/2016/1226/144430_A4hf_2945455.png" width="484"></p> 
<span id="OSC_h4_8"></span>
<h4>之后选择SDK路径</h4> 
<p><img alt="" height="361" src="https://static.oschina.net/uploads/space/2016/1226/145305_gAYV_2945455.png" width="686"></p> 
<p><img alt="" height="447" src="https://static.oschina.net/uploads/space/2016/1226/145409_tOrN_2945455.png" width="585"></p> 
<p><img alt="" height="335" src="https://static.oschina.net/uploads/space/2016/1226/145510_YQz5_2945455.png" width="481"></p> 
<span id="OSC_h4_9"></span>
<h4>点击Finish</h4> 
<p><img alt="" height="323" src="https://static.oschina.net/uploads/space/2016/1226/145603_8l7W_2945455.png" width="499"></p> 
<span id="OSC_h4_10"></span>
<h4>创建项目了</h4> 
<p><img alt="" height="781" src="https://static.oschina.net/uploads/space/2016/1226/145643_GCtQ_2945455.png" width="667"></p> 
<span id="OSC_h3_11"></span>
<h3>当下载为.exe程序时，直接双击.exe程序</h3> 
<p><img alt="" height="368" src="https://static.oschina.net/uploads/space/2016/1226/150124_uAso_2945455.png" width="470"></p> 
<p><img alt="" height="339" src="https://static.oschina.net/uploads/space/2016/1226/150320_cyGw_2945455.png" width="497"></p> 
<span id="OSC_h4_12"></span>
<h4>有ＳＤＫ就不选Android SDK即可，没有则选上，之后点击Next</h4> 
<p><img alt="" height="340" src="https://static.oschina.net/uploads/space/2016/1226/150550_m3Yx_2945455.png" width="500"></p> 
<span id="OSC_h4_13"></span>
<h4>第一地址是安装路径，自己选择即可，第二个是SDK路径，选择好后点击Next</h4> 
<p><img alt="" height="365" src="https://static.oschina.net/uploads/space/2016/1226/150833_tKGR_2945455.png" width="468"></p> 
<span id="OSC_h4_14"></span>
<h4>点击安装Install</h4> 
<p><img alt="" height="367" src="https://static.oschina.net/uploads/space/2016/1226/150919_hQx1_2945455.png" width="471"></p> 
<span id="OSC_h4_15"></span>
<h4>进度好了之后点击Next，</h4> 
<p><img alt="" src="https://static.oschina.net/uploads/space/2016/1226/151018_Sm87_2945455.png"></p> 
<span id="OSC_h4_16"></span>
<h4>点击Finsh之后就到最上面zip包的最初步骤了</h4> 
<p><img alt="" height="188" src="https://static.oschina.net/uploads/space/2016/1226/151151_4VQv_2945455.png" width="527"></p> 
<span id="OSC_h4_17"></span>
<h4>如果没有SDK时，起初选择下载SDK的话，只下载了最新版本，如果还要更新更多版本的话就点击 &nbsp;Android Studio顶部按钮&nbsp;<img alt="" height="42" src="https://static.oschina.net/uploads/space/2016/1226/152345_cnaW_2945455.png" width="43">&nbsp;&nbsp;就会出现 &nbsp;</h4> 
<p><img alt="" height="547" src="https://static.oschina.net/uploads/space/2016/1226/152522_SqRZ_2945455.png" width="794"></p> 
<span id="OSC_h4_18"></span>
<h4>选择自己要的版本，点击OK就可下载了，如果以后要更新Android Studio新版本时，点击顶部</h4> 
<span id="OSC_h4_19"></span>
<h4>File--》Setting--》如下图</h4> 
<p><img alt="" height="487" src="https://static.oschina.net/uploads/space/2016/1226/153011_vZSM_2945455.png" width="825"></p> 
<span id="OSC_h4_20"></span>
<h4>选择Dev &nbsp;Channel &nbsp; 点击Check Now &nbsp;检测新版本，如何图则可以更新 ，点击Update &nbsp;and&nbsp;Restart则更新新版本</h4> 
<p><img alt="" height="367" src="https://static.oschina.net/uploads/space/2016/1226/153328_tg2L_2945455.png" width="892"></p>
